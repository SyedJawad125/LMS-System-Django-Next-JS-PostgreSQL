# If you encounter issues:

# 1. ChromaDB issues:
#    pip install --upgrade chromadb
#    pip install hnswlib --no-cache-dir

# 2. Sentence-transformers issues:
#    pip install torch --index-url https://download.pytorch.org/whl/cpu
#    pip install sentence-transformers

# 3. PyPDF2 issues:
#    pip install --upgrade PyPDF2

# 4. psycopg2 issues (especially on Mac M1/M2):
#    brew install postgresql
#    pip install psycopg2-binary --no-cache-dir

# 5. Memory issues with embeddings:
#    Use smaller model: "all-MiniLM-L6-v2" (current)
#    Or use: "all-MiniLM-L12-v2" for better quality
#    Avoid: "all-mpnet-base-v2" (requires more RAM)
